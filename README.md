# yt-dlp-communionafterdark

yt-dlp extractor plugin for [Communion After Dark](https://www.communionafterdark.com/).

## Installation

Requires yt-dlp `2023.01.02` or above.

You can install this package with pip:

```
python3 -m pip install -U https://github.com/glensc/yt-dlp-communionafterdark/archive/master.zip
```

See [installing yt-dlp plugins](https://github.com/yt-dlp/yt-dlp#installing-plugins) for other installation methods.

## Usage

```
yt-dlp https://www.communionafterdark.com/episode-slug
```
